## 页面构成

整个 `html` 页面主要由 `<html/>` 标签，`<head/>` 标签，`<body/>` 标签组成。

```html
<!DOCTYPE HTML>
<html>

<head>
	...
</head>

<body>
	...
</body>

</html>
```

## \<head\>

`<title>` 内容将显示在浏览器 `tab` 栏上。
